Проект для объединения линтеров для разных языков.
Планируется прикрутить интерфейс, возможность добавлять свои интерфейсы и языки

Скорее всего будет запуск через докерфайл, т к бинарники на разных ОС работать не будут, сделать кроссплатформу будет очень сложно и сильно обрежет возможные варианты для развития

Сейчас пока реализован только питон, вот команды для запуска и теста:
```
python -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt 
python3 main.py python .
```

Или `python3 main.py -h` для вывода справки
